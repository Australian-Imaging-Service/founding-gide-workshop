# Founding GIDE - Australian Imaging Service Workshop

Workshop materials for the Founding GIDE - Australian Imaging Service Workshop

There are two sections of this tutorial, the first is the Pydra Tutorial under the `notebooks/pydra.ipynb` Python notebook,
and the second is a tutorial on how to deploy pipelines to the Australian Imaging Service communities repository in the `notebooks/ais-pipelines.ipynb` Bash notebook
